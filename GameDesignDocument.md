# Defense Log - 게임 기획서

## 게임 개요

**장르**: 멀티플레이어 디펜스 로그라이크
**플랫폼**: PC (Unity 6 - 6000.0.41f1)
**그래픽 스타일**: 2.5D (2D 캐릭터 스프라이트 + 3D 환경)
**플레이어 수**: 1~4명

## 게임 목표

플레이어들이 협력하여 중앙에 있는 건물을 방어하면서 성장하고, 충분히 강해지면 적진의 보스를 처치하여 게임을 클리어한다.

---

## 핵심 게임 플레이

### 1. 웨이브 디펜스 시스템
- **디펜스 단계**: 중앙 건물을 지키며 몰려오는 적 몬스터들을 처치
- **재정비 시간**: 각 웨이브 종료 후 일정 시간 동안 재정비 (30초~60초)
  - 상점 이용
  - 능력치 배분
  - 전직 결정
  - 회복 및 준비
- **웨이브 진행**: 시간이 지나면 다음 웨이브 자동 시작 (또는 수동 시작 옵션)

### 2. 경제 시스템
- **골드 획득**: 몬스터 처치 시 골드 드랍
- **중앙 건물 상점**: 재정비 시간에 상점 이용 가능
- **상점 판매 물품**:
  - **회복 물약** (HP 포션, 마나 포션)
  - **장비 아이템** (무기 강화, 방어구)
  - **능력치 초기화 주문서** - 능력치 포인트 재분배
  - **스킬 강화 주문서** - 스킬 레벨업 또는 강화
  - **소모품** (버프 아이템, 부활 아이템 등)

### 3. 성장 시스템
- **경험치**: 몬스터 처치로 경험치 획득
- **레벨업**: 레벨당 능력치 포인트 3~5점 획득
- **능력치 배분**: 플레이어가 원하는 능력치에 자유롭게 배분
- **전직 시스템**: 일정 레벨 도달 + 능력치 조건 충족 시 전직 가능
- **능력치 초기화**: 상점에서 주문서 구매로 재분배 가능

### 4. 보스 처치
- 충분히 성장하면 적진의 보스에게 도전
- 보스 처치 시 게임 클리어

## 캐릭터 시스템

### 기본 능력치
1. **STR (힘)** - 물리 공격력, 근접 데미지
2. **DEX (민첩)** - 공격 속도, 회피율, 원거리 공격력
3. **INT (지능)** - 마법 공격력, 마나량
4. **VIT (체력)** - HP, 방어력
5. **LUK (행운)** - 크리티컬 확률, 드랍률

### 1차 직업 (4종)

#### 1. 전사 (Warrior)
- **특징**: 높은 체력과 방어력, 근접 전투
- **전직 조건**: STR 20+, VIT 15+
- **주 능력치**: STR, VIT

#### 2. 궁수 (Archer)
- **특징**: 원거리 공격, 다양한 사거리 옵션
- **전직 조건**: DEX 20+, LUK 10+
- **주 능력치**: DEX, LUK

#### 3. 마법사 (Mage)
- **특징**: 강력한 마법 공격, 광역 스킬
- **전직 조건**: INT 25+
- **주 능력치**: INT, VIT

#### 4. 도적 (Thief)
- **특징**: 빠른 연속 공격, 높은 기동성, 크리티컬
- **전직 조건**: DEX 15+, LUK 15+
- **주 능력치**: DEX, LUK

### 직업별 외형 변화
- **1차 전직**: 캐릭터 외형 + 무기 모두 변경
  - 예: 초보자 → 전사 (전사 의상 + 무기 장착)
  - 예: 초보자 → 마법사 (마법사 로브 + 지팡이 장착)
  - 캐릭터 의상, 무기 변화
- **2차 전직**: 캐릭터 외형 + 무기 모두 재변경
  - 더 화려한 의상, 헤어, 이펙트
  - 더 강력하고 독특한 무기 외형

### 2차 직업 (추후 확장)
각 1차 직업마다 3개의 2차 전직 루트 제공

#### 전사 (Warrior) → 2차 전직
1. **검 (Sword)** - 빠른 검술, 연속 공격 특화
2. **둔기 (Mace)** - 강력한 단타 데미지, 스턴 특화
3. **봉 (Polearm)** - 넓은 범위 공격, 중거리 특화

#### 궁수 (Archer) → 2차 전직
1. **거너 (Gunner)** - 중거리 화기 공격, 빠른 연사
2. **활 (Bow)** - 장거리 정확한 저격, 단일 대상 고데미지
3. **석궁 (Crossbow)** - 관통 공격, 중거리 범위 데미지

#### 마법사 (Mage) → 2차 전직
1. **원소 (Elemental)** - 불/번개/얼음 원소 마법, 광역 공격
2. **성 (Holy)** - 회복 및 보호 마법, 지원 특화
3. **악 (Dark)** - 저주 및 디버프, 지속 데미지

#### 도적 (Thief) → 2차 전직
1. **격투가 (Fighter)** - 빠른 연속 타격, 근접 콤보
2. **단검 (Dagger)** - 크리티컬 특화, 기습 공격
3. **표창 (Shuriken)** - 원거리 다중 투척, 빠른 공격

## 로그라이크 요소

### 레벨 및 성장
- **경험치 시스템**: 몬스터 처치로 경험치 획득
- **레벨업**: 레벨당 능력치 포인트 3~5점 획득
- **자유 성장**: 플레이어가 원하는 능력치에 자유롭게 배분

### 전직 시스템
- 일정 레벨 도달 + 능력치 조건 충족 시 전직 가능
- 전직 시 고유 스킬 및 특성 해금
- 매 플레이마다 다른 빌드 시도 가능

### 사망 패널티
- **사망 시**: 일정 시간(30초~60초) 후 부활
- **패널티**: 레벨 및 경험치 초기화 (능력치도 초기화)
- **건물 HP**: 플레이어가 죽어있는 동안 건물이 더 위험해짐

## 멀티플레이어

### 협동 플레이
- 1~4명의 플레이어가 함께 플레이
- 플레이어 수에 따라 적 난이도 자동 조정
- 팀워크가 중요: 역할 분담 (탱커, 딜러, 서포터 등)

### 역할 분담 예시
- **전사**: 앞에서 적을 막고 건물 보호
- **궁수**: 후방에서 안전하게 화력 지원
- **마법사**: 적 무리를 광역으로 제거, 지원 마법
- **도적**: 빠르게 이동하며 위험한 적 우선 처리

## 맵 구조 및 방어 시스템

### 맵 레이아웃

```
[적 스폰 지역]
        ↓
        ↓ (일반 몹 + 엘리트 몹 + 보스)
        ↓
    [전투 지역]
    (미니언 + 플레이어 vs 적들)
        ↓
        ↓ (적이 돌파하면)
        ↓
    [포탑 방어선] ← 마지막 방어선
        ↓
   ┌─────────────────────┐
   │   [중앙 건물 지역]   │
   │                     │
   │  ● 중앙 건물 (Core) │ ← HP 0 = 패배
   │                     │
   │  주변 기능 건물:    │
   │  • 상점            │
   │  • 전직소          │
   │  • 포탑 강화소     │
   │  • 미니언 막사     │
   └─────────────────────┘
        ↓
 [플레이어 부활 지점]

[보스 지역] ← 성장 후 도전 (최종 웨이브)
```

### 방어 시스템 (3단계)

#### 1단계: 미니언 방어 (최소한의 방어)
- **역할**: 적의 진행 속도 늦추기, 시간 벌기
- **전투력**: 일반 몹보다 약함 (1:3 정도로 밀림)
- **중요성**: 플레이어가 전투 위치로 이동할 시간 확보
- **특징**:
  - 중앙 건물에서 자동 소환
  - 적 스폰 지역 방향으로 자동 이동
  - 적을 발견하면 교전
  - 사망 시 일정 시간 후 재소환

#### 2단계: 플레이어 방어 (핵심 전투력)
- **역할**: 미니언과 함께 적 처치
- **중요성**: 게임의 핵심, 플레이어가 없으면 방어 불가
- **전략**:
  - 일반 몹: 미니언과 협력하여 처치
  - 엘리트 몹: 우선 타겟팅, 빠르게 제거
  - 보스: 집중 공격 필요
- **위험**: 플레이어가 멀리 떨어지면 미니언만으로는 막을 수 없음

#### 3단계: 포탑 방어 (최후의 안전망)
- **역할**: 마지막 방어선, 뚫고 온 적 자동 공격
- **설치**: 중앙 건물 주변 고정 위치에 설치
- **제한**: 설치 개수 제한 (골드 필요)
- **업그레이드**: 포탑 강화소에서 강화 가능

### 패배 조건

**중앙 건물 HP 0 → 게임 오버**

시나리오:
1. 미니언이 적을 잠시 붙잡음
2. 적이 미니언 돌파
3. 플레이어가 방어 실패 또는 다른 곳에 있음
4. 포탑이 일부 적 처치
5. 남은 적들이 중앙 건물 공격
6. 건물 HP 0 → **패배**

## 건물 시스템

### 중앙 건물 (Core Building)
- **HP**: 1000~3000 (난이도에 따라)
- **기능**:
  - 미니언 자동 소환 (30초마다)
  - 게임 허브 역할
  - 파괴 시 게임 오버
- **상호작용**: 없음 (방어 대상)

### 기능 건물 (5종)

모든 기능 건물은 **우클릭으로 상호작용**:
- 멀리서 우클릭 → 플레이어가 자동으로 이동 → 도착 시 UI 자동 열림
- 가까이서 우클릭 → 즉시 UI 열림
- **웨이브 중에도 사용 가능** (긴급 회복, 포탑 추가 등)

#### 1. 상점 (Shop)
- **위치**: 중앙 건물 근처
- **기능**:
  - HP 포션 구매
  - 마나 포션 구매
  - 장비 아이템 구매
  - 능력치 초기화 주문서
  - 버프 소모품
- **결제**: 골드

#### 2. 전직소 (Job Hall)
- **위치**: 중앙 건물 근처
- **기능**:
  - 1차 전직 (레벨 10 + 능력치 조건)
  - 2차 전직 (레벨 30 + 능력치 조건)
  - 직업 정보 확인
  - 전직 조건 확인
- **결제**: 무료 (조건만 충족하면)

#### 3. 포탑 강화소 (Turret Workshop)
- **위치**: 중앙 건물 근처
- **기능**:
  - 포탑 신규 설치 (설치 위치 선택)
  - 포탑 업그레이드 (데미지, 사거리, 공격 속도)
  - 포탑 수리
  - 포탑 철거 및 재배치
- **결제**: 골드
- **제한**: 최대 포탑 수 제한 (5~10개)

#### 4. 미니언 막사 (Minion Barracks)
- **위치**: 중앙 건물 근처
- **기능**:
  - 미니언 능력치 업그레이드 (공격력, 체력, 이동속도)
  - 미니언 소환 속도 증가
  - 동시 존재 가능한 미니언 수 증가
  - 미니언 타입 변경 (탱커/딜러/서포터 - 추후 확장)
- **결제**: 골드
- **효과**: 모든 미니언에 적용

#### 5. 능력치 배분소 (Stat Distribution)
- **위치**: 중앙 건물 근처
- **기능**:
  - 남은 능력치 포인트 배분
  - 능력치 정보 확인
  - 현재 파생 스탯 확인
- **결제**: 무료 (포인트만 있으면)

## 적 시스템 및 밸런스

### 적 종류

#### 1. 일반 몹 (Normal Enemy)
- **전투력**: 미니언보다 강함 (미니언 3마리 = 일반 몹 1마리)
- **특징**:
  - 기본 체력 및 공격력
  - 미니언을 빠르게 돌파
  - 플레이어가 직접 처리해야 함
- **스폰**: 웨이브마다 대량 스폰
- **보상**: 경험치 + 골드

#### 2. 엘리트 몹 (Elite Enemy)
- **전투력**: 일반 몹의 3~5배
- **특징**:
  - 높은 체력
  - 강한 공격력
  - 미니언으로는 절대 막을 수 없음
  - 특수 능력 보유 가능 (빠른 이동속도, 광역 공격 등)
- **스폰**: 웨이브 중반부터 등장, 후반으로 갈수록 증가
- **보상**: 경험치 2배 + 골드 3배
- **시각적 구분**: 크기 크거나 특수 이펙트

#### 3. 보스 (Boss)
- **전투력**: 엘리트의 10배 이상
- **특징**:
  - 매우 높은 체력
  - 강력한 공격 패턴
  - 특수 스킬 보유
  - 미니언 + 포탑으로도 막기 어려움
- **스폰**: 최종 웨이브 또는 특정 웨이브
- **보상**: 대량 경험치 + 골드 + 특별 아이템
- **난이도**: 플레이어 수에 따라 조정

### 전투력 비교표

```
보스          ████████████████████ (100)
엘리트 몹     ████████              (40)
일반 몹       ███                   (15)
미니언        █                     (5)
```

### 웨이브별 적 구성

**웨이브 1-3 (초반):**
- 일반 몹 위주 (5~10마리)
- 엘리트 없음
- 미니언 + 플레이어로 여유롭게 방어

**웨이브 4-6 (중반):**
- 일반 몹 증가 (10~20마리)
- 엘리트 1~2마리 등장
- 미니언이 시간 벌어주는 동안 플레이어가 엘리트 우선 처리

**웨이브 7-9 (후반):**
- 일반 몹 대량 (20~30마리)
- 엘리트 3~5마리
- 미니언 빠르게 돌파, 플레이어 정신없이 방어
- 포탑이 중요해짐

**웨이브 10 (최종):**
- 일반 몹 + 엘리트 + **보스 등장**
- 총력전
- 미니언, 포탑, 플레이어 모두 동원

### 적 AI 우선순위

```
1순위: 플레이어 (감지 범위 내)
2순위: 미니언 (교전 중)
3순위: 중앙 건물 (최종 목표)
```

## 게임 진행 흐름

### 전체 흐름
1. **게임 시작** → 플레이어들 중앙 건물 근처에서 시작
2. **웨이브 1~3 (초반)** → 약한 적, 레벨업 및 골드 획득
3. **재정비 시간** → 상점 이용, 능력치 배분
4. **웨이브 4~6 (중반)** → 중간 난이도 적, 1차 전직 가능
5. **재정비 시간** → 전직, 장비 구매, 스킬 강화
6. **웨이브 7~10 (후반)** → 강한 적, 본격적인 성장
7. **최종 웨이브** → 보스 출현 또는 보스 지역으로 이동
8. **보스 처치** → 게임 클리어!

### 웨이브 구조
```
[웨이브 1] → [재정비 60초] → [웨이브 2] → [재정비 60초] → ...
     ↓              ↓                ↓              ↓
  몬스터 처치    상점 이용        몬스터 처치    능력치 배분
  경험치 획득    회복             골드 획득      전직
```

- 상점에서 물약, 장비 구매
- 능력치 포인트 배분 또는 초기화
- 스킬 레벨업 및 강화

## 조작 시스템

### 기본 조작 (RTS/MOBA 스타일)
게임은 리그 오브 레전드, 스타크래프트와 유사한 조작 방식을 사용합니다.

### 마우스 조작
- **우클릭 (Right Click)**: 통합 명령
  - **지형** 우클릭 → 해당 위치로 이동
  - **적** 우클릭 → 추적하며 공격
  - **기능 건물** 우클릭 → 이동 후 자동 상호작용 (UI 열림)
    - 멀리 있을 때: 건물로 이동 → 도착 시 UI 자동 열림
    - 가까이 있을 때: 즉시 UI 열림
    - 웨이브 중에도 사용 가능

- **좌클릭 (Left Click)**: 스킬/아이템 사용 확정
  - 스킬 키를 누르면 범위 표시
  - 좌클릭으로 대상/방향 지정 후 발동

### 키보드 조작

#### 공격 및 스킬
- **Q**: 기본 공격
- **W, E, R**: 주요 스킬 (직업별 고유 스킬)
- **A, S, D, F**: 추가 스킬 또는 보조 스킬

#### 아이템
- **1, 2, 3, 4**: 소모 아이템 (포션, 버프 아이템 등)

### 스킬 사용 방식
1. 스킬 키를 누름 (예: W)
2. 화면에 스킬 범위/사거리가 표시됨
   - 원형 범위 (AOE 스킬)
   - 직선/부채꼴 (방향성 스킬)
   - 대상 지정 (단일 타겟 스킬)
3. 마우스 좌클릭으로 확정
4. 스킬 발동

### 스킬 타입 예시
- **즉시 발동형**: 키를 누르면 즉시 자신 주변에 발동 (버프, 자가 회복 등)
- **범위 지정형**: 원형 범위를 표시하고 좌클릭으로 위치 지정
- **방향 지정형**: 직선/부채꼴 범위를 표시하고 좌클릭으로 방향 지정
- **대상 지정형**: 사거리 내 적을 좌클릭으로 선택

## 스킬 시스템

### 직업별 고유 스킬
각 직업은 기본 공격(Q) + 고유 스킬 3~5개를 보유합니다.

#### 전사 (Warrior) - 스킬 예시
- **Q (기본 공격)**: 전방 근접 공격
- **W (돌진)**: 적에게 돌진하여 데미지 + 스턴
- **E (방어 태세)**: 일정 시간 받는 데미지 감소
- **R (회전 베기)**: 주변 적들에게 광역 데미지

#### 마법사 (Mage) - 스킬 예시
- **Q (기본 공격)**: 마법 탄환 발사
- **W (파이어볼)**: 범위 지정 화염 폭발
- **E (아이스 스파이크)**: 직선 관통 냉기 공격
- **R (메테오)**: 광역 궁극기, 큰 범위에 낙하 공격

#### 궁수 (Archer) - 스킬 예시
- **Q (기본 공격)**: 단일 대상 화살 발사
- **W (멀티샷)**: 부채꼴 범위 다중 화살
- **E (후방 도약)**: 뒤로 점프하며 회피
- **R (관통의 화살)**: 직선 관통 강력한 화살

### 스킬 강화
- **스킬 레벨업**: 레벨업 시 스킬 포인트 획득
- **스킬 강화 주문서**: 상점에서 구매하여 추가 강화
- **강화 효과**: 데미지 증가, 쿨타임 감소, 범위 증가 등
